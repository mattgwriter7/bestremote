The "Hey Now!" is an <h1> tag. It looks all fancy because some CSS is applied to it. These "main heading" tags should be used judiciously. (I am only using them at the start of "panels," which I explain in a moment.)

Panels

A "div.panel" is like a section that can be shown or hidden. They are denoted by a lot of vertical padding, plus the little black dots on top and bottom. (The content between the dots is the "panel.")

Panels always start with a "div.interior" which is used to actually add the padding.

This section on panels starts with an <h2> tag, which (obviously) is a bit smaller than an <h1> tag. I don't use any headings besides h1 and h2.
NOTE: H1s and H2s may have different margins and paddings when used in a grid. "Grids" are explained below.
